# StitchClone — AI UI 生成器

> 🎨 基于自然语言的 AI 驱动 UI 生成器，专注中国电商场景（双11、618、活动落地页）

## 产品概述

StitchClone 是一个 AI 原型的 UI 生成工具，用户输入自然语言描述，即可生成生产级 HTML/Tailwind 代码，并实时预览效果。

**灵感来源：** Google Stitch 2.0（Product Hunt March 2026 #1 AI 产品）

## 核心功能

- 🗣️ **自然语言输入** — 用中文描述你想要的页面，系统解析并生成代码
- 🎨 **实时预览** — 生成后立即在右侧预览，支持桌面/平板/手机三种尺寸
- 📋 **一键复制/下载** — 生成 HTML 代码一键复制，或下载为文件
- 🏷️ **快速组件库** — 点击即可添加：主图Banner、商品卡片、倒计时、优惠券、评价星级等
- 📱 **电商场景覆盖** — 内置手机促销、双11、618 三大场景模板
- ⚡ **响应式预览** — 自适应桌面、平板、手机三种视口

## 技术栈

| 层次 | 技术 |
|------|------|
| 前端框架 | 原生 HTML + Tailwind CSS (CDN) |
| 图标 | Emoji (无额外依赖) |
| 字体 | Inter (Google Fonts) |
| 部署 | 纯静态，可直接打开 HTML 或部署到任意 CDN |

## 快速开始

1. 打开 `index.html`
2. 在左侧输入描述词，或点击模板按钮填充
3. 点击 **「生成 UI」** 按钮
4. 在右侧预览效果
5. 复制代码或下载 HTML 文件
